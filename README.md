# Buenas Viagens

Projeto desenvolvido para a Avaliação III da disciplina Tecnologias de Internet.

## Descrição

Landing page de uma agência de viagens fictícia chamada Buenas Viagens.

O projeto foi desenvolvido a partir de um layout de referência no Figma, com estrutura em HTML5 e estilização em CSS3.

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Git/GitHub
- Figma

## Funcionalidades

- Cabeçalho com menu de navegação
- Banner principal
- Formulário de busca
- Seção de passeios e destinos
- Cards de destinos turísticos
- Seção de avaliações de clientes
- Rodapé com informações institucionais
- Newsletter
- Efeitos de hover em links e botões
- Layout responsivo
